<h1 align="center">
    <br>WhatsBot<br/>
</h1>

A simple bot to send messages on whatsapp

## Installation
```sh
  $ cd whatsBot
  $ pip install selenium
  $ pip install webdriver_manager
```

## Usage
Enter the name of the contacts or groups you want to send a message to (line 10)
```sh
  $ python bot.py
```
